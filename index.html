Comandos utilizados:

git init
Iniciar um repositório no computador

git add .
Adicionar as alterações dos arquivos

git status
Verificar as alterações que ocorreram

git commit -m "Texto do commit"
Explicar o que está sendo adicionado

git push origin main
Enviar as alterações para o repositório da origem

git pull
Baixar alterações do repositório da origem
